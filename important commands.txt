Yes. In DB-First scaffolding:

* `Microsoft.EntityFrameworkCore.Tools`
  → provides scaffold command

* `Microsoft.EntityFrameworkCore.Design`
  → generates the `Models` and `DbContext`

* `Microsoft.EntityFrameworkCore.SqlServer`
  → reads SQL Server database schema and connects to DB.
